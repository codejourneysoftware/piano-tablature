  • • 5-Finger Home Base:
    5 = Root (lowest), 4 = +2, 3 = +4, 2 = +5, 1 = +7 (highest semitone)
  • • Chromatic Accidentals:
    Append # (sharp +1 semitone) or b (flat -1 semitone)
  • • Hand Stretches (Capped at ±2):
    • Append + (stretch up +1) or ++ (stretch up +2)
    • Append - (stretch down -1) or -- (stretch down -2)
  • • The Cross vs. The Stretch:
    • The Stretch: Reaching a finger slightly outward while keeping the hand in the same position (use +, ++, -, --).
    • The Cross: Pivoting your hand (thumb under or fingers over) to move smoothly up/down. Digitally represented by inserting a new Hand Pos marker (e.g. G Pos) in the grid to shift your home base.
  • • Multiple Lyric Verses:
    To display multiple verses sharing the same chords and fingering, pass a 2D array of arrays to the "lyrics" property (e.g., "lyrics": [ [ { "text": "Verse 1 lyrics", "col": 0 } ], [ { "text": "Verse 2 lyrics", "col": 0 } ] ]).
  • ⚠️ Boundary Rule:
    Stretches are physically capped at 2 semitones. For further notes, enter a hand position change (e.g. G Pos) in the grid instead!
